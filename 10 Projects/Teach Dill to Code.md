# Teach Dill to Code

**Status:** Planning Phase
**Student:** Dylan (Dill)
**Start Date:** 2024-02-28

## Current Assessment
Dylan is already proficient in Scratch. Ready to transition to text-based programming.

## Language Options Evaluated

### Lua (Considered, Vetoed)
**Pros:**
- Simple syntax
- Good game engines (love2d, pico-8)
- Lightweight and fast

**Cons:**
- Lil's veto (fair concern about age-appropriateness)

### Python (Selected)
**Tool:** [Thonny IDE](https://thonny.org/)

**Pros:**
- Comes with Python 3.10 environment - no setup hassle
- Variable inspection built-in
- Has PIP GUI for installing packages
- Nice integration with p5 library for graphics
- Industry-standard language
- Large community and resources

**Cons:**
- Not the prettiest interface
