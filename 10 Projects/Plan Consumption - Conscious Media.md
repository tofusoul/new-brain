# Plan Consumption - Conscious Media

**Status:** Idea Phase
**Goal:** Be more intentional about media consumption instead of watching content randomly

## Overview
Develop a system for conscious consumption of media and content rather than "willy nilly" viewing habits. Focus on quality over quantity.

## Approach
- Track what I want to consume
- Create a queue/list of content
- Make intentional choices about what to watch/read
- Avoid aimless browsing and random content discovery

## Tasks

### Planning Phase
- [ ] #task Define criteria for valuable content
- [ ] #task Create content queue system (could be spreadsheet, app, or simple list)
- [ ] #task Decide on categories to track (books, movies, shows, articles, videos)
- [ ] #task Set up tracking mechanism

### Implementation
- [ ] #task Fill queue with pre-vetted, high-quality content
- [ ] #task Establish habit of checking queue before browsing
- [ ] #task Review and refine queue regularly
- [ ] #task Track consumption patterns

### Review
- [ ] #task Evaluate if conscious consumption feels better
- [ ] #task Adjust system based on what works
- [ ] #task Share findings if valuable

## Potential Tools
- Obsidian note for tracking
- Dedicated spreadsheet
- Notion/Airtable database
- Simple text file
