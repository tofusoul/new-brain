# Zeotrope - Handmade Animation Tool

**Status:** Idea Phase
**Type:** Art/Craft Project with Software Component

## Overview
Make handmade zeotropes and a simple app that cuts the right number of frames out of a video and makes them into printable pictures.

## Why Zeotropes?

### Reasons
- It's pretty fun and still magical to see real-world things come to life
- You can present GIFs in a cool way without complex electronics
- Handmade objects have a unique aesthetic
- Low-tech animation has charm

## How to Do It

### Research Phase
- Zeotropes are well-known devices
- Building them requires only a little bit of research

### Software Component
**Challenge:** Making software so you can print your own animation snippets will be hard

**Features Needed:**
- Import video
- Select number of frames
- Export printable images
- Calibrate for physical zeotrope dimensions

### Open Source
Make everything open-source to welcome improvements.

## Business Model

### Local Markets
- Build zeotropes to sell at local markets
- Find feedback and iterate
- Connect with community

### Low Risk
- If they don't sell, it's still an easy and cool project
- Learning experience regardless of commercial success

## First Prototype

### Specifications
- Make each frame about the size of a quarter of a standard photo paper
- Test if the magic is still there for people
- Iterate on size and frame count based on testing
