## 🎯 System Design Notes

### What Andrew Needs
- **Fast capture** - offload thoughts in seconds (Essential Space does this)
- **Single place** - not scattered across apps (Obsidian as hub)
- **At-a-glance view** - what do I do NOW? (dashboard view)
- **Emotional prioritization** - what matters RIGHT NOW (not logical priority)
- **Handle interruptions** - kids throw you off, recover fast (pause/resume)
- **Trust system** - won't miss important things (reliable tracking)
- **Fun/accessible** - actually use it (gamification, small wins, visual)

### System Being Built
1. **Capture:** Essential Space (phone) → fast, anywhere
2. **Process:** Regularly review → move to Obsidian
3. **Organize:** Obsidian tags, links, folders
4. **Prioritize:** Daily/weekly review → what NOW?
5. **Execute:** Habit system, project tracking
6. **Track:** This note → what Tofurky is doing/questions

### Key Principles
- One question at a time (AuDHD brain can't handle multiple)
- Direct over encouraging (clear feedback, not sugarcoating)
- Proactive questions (ask when need clarity)
- Systems thinking (life as code to be refactored)
- Externalize mental load (reduce overwhelm)
