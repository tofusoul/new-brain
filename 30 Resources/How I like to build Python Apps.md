- [Python](./Python.org)

- understand the source data by:

- Collecting them into a data lake

- Parse and persist them into parquet files

- which will take care of some of the basic typing problems

- shape the data with [Pandas](./Pandas.org) as a bit of a swiss army knife

- Model the data with frozen dataclasses

- which allows you to interact with it in a largely functional programming way.

- Mock the ui and interactions with [Streamlit](./streamlit.org)

- persist the data with [DuckDB](./DuckDB.org) by mapping the model to the database
