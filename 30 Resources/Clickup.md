Another productivity app, seems to have the benefit of hindsight compared with trello etc.
- going to try it [here](https://app.clickup.com)


**Some Thoughts for Use in Enviroflow**
    - it's good if we use the added features. 
    - ~~tags didn't get imported~~
	- they do get imported, just not when the method is 'list'
    - boards are less seperate than in trello
    - can use relationships to model shared drain works
	- relationships aren't visually stand out. but can easily link from one task to another
	- should probably creat project cards for shared drain projects to link that way, cleaner
    - can use relationships and gannt charts to do what float does
    - there's no getting around manual cleanup of the data if shifting over
	- probably have to do a couple of training sessions
    - won't get the full benefits of the shift if we use the product as we would trello
    - Clickup is slow like trello with this amount of data
    - maybe could migrate with the data in the enviroflow app. 
    - there are some quirks with clickups that are weird too
	- e.g. imported number not rendering
	- added relationship fieldd not showing up as a custom field
	- tags aren't very friendly
    - permission settings are good, probabaly the biggest up
    - the list idea in trello could be either list, or status, deciding what that means has a real impact on what how the data is modeled
	- it's an extra complexity and thing to consider, trello is conceptually simpler
	- some of this can be demonstrated with a csv import
