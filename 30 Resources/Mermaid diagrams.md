A way to use plain text to declaritively create diagrames. 
can be embedded in in a lot of [[Markdown]] engines

- #note to setup syntax highlighting in [[Neovim]] markdown `TSInstall markdown-inline`
    - also, to install for standalone mermaid diagrames `TSinstall mermaid`
