# Task Workflow Guide

**Quick reference for managing tasks with confidence.**

---

## 📥 Capture (Anytime)

**Via Telegram:** Send me a task → I add to `Inbox/DAILY.md`

**In Obsidian:** Add task anywhere with `- [ ] #task Task name`

---

## 🗂️ Triage (Morning Routine)

### Step 1: Check Today's Dashboard
Open your daily note — see what's already scheduled/due

### Step 2: Process Inbox
Look at the Inbox section in [[30 Resources/Task Dashboard]]

For each task, choose:

**Today?**
- Add `⏳ 2026-MM-DD` (scheduled for today)
- Optionally add `🔺` or `⏫` for high priority

**This week?**
- Add `📅 2026-MM-DD` (due date)
- Add priority if needed

**Someday/Maybe?**
- Add tag `#backlog` or `#someday`
- Keep in appropriate Area note

**Project-specific?**
- Move to project note in `10 Projects/`
- Add due/scheduled dates

---

## 🏃 Daily Workflow

### Morning
1. Review Today's Dashboard in daily note
2. Pick 1-3 tasks to focus on (use the `⌛✨ Now` section)
3. Start with highest priority or hardest task

### Throughout the day
- Capture new tasks via Telegram (fastest) or directly in Inbox
- Mark tasks complete when done `[x]`

### Evening
1. Review what got done ✅
2. Move uncompleted tasks to tomorrow (update scheduled date)
3. Check full dashboard for anything slipping through

## 🗿 Quick Task Reference

### Date Signifiers
- `📅 2026-02-23` — Due date (deadline)
- `⏳ 2026-02-23` — Scheduled date (when you plan to work on it)
- `🛫 2026-02-23` — Start date (earliest you can work on it)

### Priority Signifiers
- `🔺` Highest
- `⏫` High
- `🔼` Medium
- `(no signifier)` Normal
- `🔽` Low
- `⏬️` Lowest

### Recurrence
- `🔁 every day`
- `🔁 every week`
- `🔁 every month`

## 📊 Dashboard Views

- **Today:** Your daily note — what's on deck

- **By Project:** Project notes contain filtered task lists
- **By Area:** Area notes contain relevant tasks

---

## ✨ Golden Rules

1. **Everything gets a date** — or it's invisible
2. **One place to look** — Today's Dashboard in daily note
3. **Priority matters** — Not everything is 🔺
4. **Capture fast, triage slow** — Don't slow down capture
5. **Trust the system** — If it's dated, you'll see it when it matters

---

## 🔄 System Feedback

**If the system feels broken:**
- Too many overdue → You're overcommitting. Say no more.
- Inbox piling up → Triage daily, or lower triage frequency.
- Tasks feel urgent all the time → Re-evaluate priorities.

**If the system feels easy:**
- You're doing it right 🗿
