# Beverages Collection Guide

## Overview
This collection documents beverages including beers, wines, spirits, and non-alcoholic drinks. Each entry includes detailed information about the beverage, tasting notes, and personal ratings.

## Collection Structure

### Categories
- **Beer**: Ales, lagers, stouts, IPAs, craft beers
- **Wine**: Red, white, rosé, sparkling
- **Spirits**: Whiskey, vodka, gin, rum, tequila
- **Non-Alcoholic**: Coffee, tea, juices, specialty drinks

### Key Attributes
- **Name**: Beverage name and brand
- **Type**: Specific style or variety
- **Category**: Broad classification (beer, wine, etc.)
- **Producer**: Brewery, winery, or manufacturer
- **Rating**: Personal 1-5 star rating
- **ABV**: Alcohol by volume percentage
- **Serving Temperature**: Optimal serving temp (cold, room temp, hot)
- **Ingredients**: Key ingredients and components
- **Flavor Profile**: Taste characteristics and notes
- **Location**: Where acquired or consumed
- **Price**: Cost information
- **Notes**: Personal observations and reviews

## How to Add New Beverages

1. Create a new note in the appropriate category folder
2. Use the beverage template structure
3. Include image if available
4. Add tasting notes and personal rating
5. Link to producer or related beverages
