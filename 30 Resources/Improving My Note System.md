# Improving My Note System

## How Obsidian Works

Obsidian is a note-taking application that stores notes as local Markdown files. This means you have complete control over your data. It uses a "vault," which is a folder on your computer where all your notes are kept.

The key features of Obsidian are:

*   **Bi-directional linking:** You can link notes to each other using the `[[Note Name]]` syntax. This creates a network of connected thoughts.
*   **Graph view:** This visualizes the connections between your notes, helping you see relationships between ideas.
*   **Plugins and themes:** Obsidian is highly customizable. You can add new features with plugins and change the look and feel with themes.

## How Your Plugins Work

Here's a summary of how the plugins you're using work:

*   **Tasks:** This plugin allows you to manage tasks within Obsidian. You can create tasks using the `- [ ]` syntax and add due dates, priorities, and recurrence. The plugin provides a powerful query language to create dynamic task lists.
*   **Day Planner:** This plugin helps you time-block your day. You can create a schedule in your daily note, and the plugin will display it in a timeline view. It can also integrate with external calendars.
*   **Calendar:** This plugin provides a calendar view in the sidebar. You can use it to navigate your daily notes and visualize your activity over time.
*   **Dataview:** This plugin turns your Obsidian vault into a database. You can add metadata to your notes and then use a query language to create dynamic lists, tables, and reports.
*   **Outliner:** This plugin improves the experience of working with lists in Obsidian. It allows you to collapse and expand list items and focus on specific parts of a list.
*   **Templater:** This plugin allows you to create and use templates for your notes. This can help you standardize the structure of your notes and automate repetitive tasks.
*   **Local REST API:** This plugin allows you to interact with Obsidian from other applications. You can use it to create, read, and update notes programmatically.
*   **Advanced URI:** This plugin allows you to create links that open specific notes or perform specific actions in Obsidian. This can be useful for creating workflows and integrating with other applications.
