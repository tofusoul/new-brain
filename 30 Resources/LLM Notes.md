# Pillars framework

**Persona** : respond as a...
**Intent** : Your goal is...
**Layout**: Format the response as...
**Limits**:  Avoid...
**Audience**: this is intended for...
**Requirements**: Ensure that you include...
**Style** : Use tone that is...
