# EGO Power+ LM2000E Lawn Mower

## Overview
- **Brand:** EGO Power+
- **Model:** LM2000E
- **Type:** 56V Lithium-ion Cordless Lawn Mower
- **Year:** ~2015 (award stickers from 2015 visible)

| Category | Detail |
|-----------|---------|
| Voltage | 56V |
| No-Load Speed | 3300/min |
| Cutting Height | 25mm to 80mm |
| Cutting Height Positions | 5 |

| Grass Bag Volume | 60 Litres |
| Mower Weight | 26.2kg (without battery pack; with grass bag) |
| Blade Model | AB2000 |

---

## Key Features
- Arc Lithium 56V battery technology
- Brushless motor
- Variable-speed self-propelled (LM2000-S variant)
- Petrol-matching power without noise, fumes or fuss
- Mulching capability
- Side discharge option

## Blade Maintenance

### Blade Specifications
| Property | Value |
|----------|-------|
| Model | AB2000 |
| Type | 3-in-1 (mulching, bagging, side discharge) |
| Size | 49cm / 20 inches |

### Torque Specifications
| Specification | Value |
|--------------|-------|
| Blade Bolt | 36-43 ft-lb (50-60 N.m) |
| Socket Size | 9/16" (14mm) |
| Direction | Clockwise to tighten |

### How Often to Sharpen
- **Light use:** Once per year
- **Heavy use:** Every 6-8 mows (approximately monthly)
- Keep a spare blade to rotate and reduce downtime
