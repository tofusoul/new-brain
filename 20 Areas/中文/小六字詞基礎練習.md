# 小學六年級字詞基礎練習

## 第1週：常用成語學習與應用

### 成語故事閱讀

#### 成語一：畫蛇添足
**故事**：
古代楚國有個人，得到一壺酒。大家商量說：「這壺酒不夠大家喝，不如我們各自在地上畫蛇，誰先畫好，誰就喝這壺酒。」

有個人畫得很快，第一個畫好了蛇。他看見別人還沒畫完，就左手拿著酒壺，右手又繼續畫，說：「我還能給蛇添上腳呢！」

還沒等他畫完，另一個人已經畫好了蛇，搶過酒壺說：「蛇本來就沒有腳，你怎麼能給它添上腳呢？」說完就把酒喝了。

**寓意**：做了多餘的事情，不但沒有好處，反而把事情搞砸了。

**造句練習**：
1. 這篇文章已經很完整了，再加這些內容就是______。
2. 他本來想幫忙，結果______，把事情弄得更糟。
3. 做事情要適可而止，避免______的情況。

#### 成語二：井底之蛙
**故事**：
一隻青蛙住在一口淺井裡。它覺得自己是世界上最快樂的生物，因為這口井就是它的整個世界。

有一天，一隻從海邊來的大龜路過井邊。青蛙驕傲地對大龜說：「你看我多快樂！我想跳就在井欄上跳，想休息就在井壁的磚縫裡休息。在這裡，我就是國王！」

大龜聽了，就向青蛙描述大海的樣子：「大海比你的井大上千萬倍，用千里也形容不了它的遼闊，用千丈也測量不了它的深度。」

青蛙聽得目瞪口呆，這才知道原來世界這麼大，自己是多麼渺小。

**寓意**：比喻見識短淺、眼界狹窄的人。

**造句練習**：
1. 我們要多讀書，多看世界，不要成為______。
2. 他從未出過遠門，真是個______。
3. 與其當______，不如走出去看看外面的世界。

#### 成語三：守株待兔
**故事**：
古代宋國有個農夫，正在田裡耕作。突然，一隻兔子慌慌張張地跑過來，一頭撞在田邊的樹樁上，摔斷脖子死了。

農夫高興地撿起兔子，回家美美地吃了一頓。從此以後，他不再耕田，整天守在那個樹樁旁邊，希望還能有兔子撞死在樹樁上。

結果，他等啊等，一天天過去，再也沒有兔子撞死在樹樁上。而他的田地因為沒人耕種，雜草叢生，莊稼都荒蕪了。

**寓意**：比喻死守經驗，不知變通，或妄想不勞而獲。

**造句練習**：
1. 學習要主動努力，不能______，等待好運降臨。
2. 他總是______，希望彩票中獎，卻從不努力工作。
3. 與其______，不如積極行動去創造機會。

### 成語接龍遊戲

#### 遊戲規則
1. 用前一個成語的最後一個字作為下一個成語的第一個字
2. 成語必須是正確的、常用的
3. 可以同音不同字

#### 練習題組
**第一組**：
畫蛇添足 → 足智多謀 → 謀事在人 → 人山人海 → 海闊天空

**第二組**：
井底之蛙 → 蛙鳴狗叫 → 叫苦連天 → 天馬行空 → 空前絕後

**第三組**：
守株待兔 → 兔死狐悲 → 悲天憫人 → 人定勝天 → 天長地久

### 成語應用練習

#### 情境選擇題
**情境一**：
小明寫作文時，已經把要說的話都表達清楚了，但又覺得字數不夠，就加了很多不必要的詞句。老師評語說：「這就是典型的畫蛇添足。」

**問題**：老師說「畫蛇添足」是什麼意思？
A. 寫得很好，很完整
B. 做了多餘的事情，反而不好
C. 字數不夠，需要補充
D. 畫畫技術很好

**情境二**：
小華從未出過國，覺得自己的城市就是全世界最好的地方。當同學們討論外國的文化時，她總是說：「那些地方哪有我們這裡好？」

**問題**：小華的行為可以用哪個成語形容？
A. 畫蛇添足
B. 井底之蛙  
C. 守株待兔
D. 一心一意

### 成語故事創作

#### 創作練習
請選擇一個成語，編寫一個現代版的故事：

**選擇題目**：
1. 現代版的「畫蛇添足」
2. 校園版的「井底之蛙」
3. 職場版的「守株待兔」

**創作要求**：
- 故事情節要符合成語寓意
- 人物設定要現代化
- 故事要有開頭、發展、高潮、結尾
- 字數控制在300-500字

**範例框架**：
```
標題：現代版畫蛇添足

人物：小美（學生）、老師、同學

情節：
開頭：小美做美術作業，畫了一幅很美的畫
發展：她覺得畫還不夠完美，不斷添加細節
高潮：添加太多細節後，畫反而變得雜亂
結尾：老師點評，讓她明白畫蛇添足的道理
```
