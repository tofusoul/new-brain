- Different ways to calculate GST. How to calculate GST.

- GST-exclusive amounts and GST-inclusive amounts can be calculated in a number of

- different ways, using the guide below can help you easily calculate it yourself:

- - GST-\*exclusive amount \*× 15% ( or 0.15, or  3/20 ) = GST.

-   GST-\*exclusive amount \*× 115% ( or 0.15, 23/20  ) = GST-**inclusive amount**.
-   GST-\*inclusive amount \*× 3/23 = GST.
-   GST-\*inclusive amount \*× 20/23 = GST-**exclusive amount**.
